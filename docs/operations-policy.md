# Operations Policy

## Payment

앱 선결제를 기본으로 합니다. 현장결제는 노쇼, 재고 보류 손실, 수수료 회수 문제를 만들 수 있습니다.

## Settlement

- 픽업 완료 주문만 정산 대상
- 플랫폼 수수료와 PG 수수료 차감
- 주 1회 판매자 정산
- 관리자 웹에서 정산 보류/환불 조정 가능

## Refund

- 픽업 시작 1시간 전까지 소비자 취소 가능
- 픽업시간 이후 미수령 시 환불 불가
- 매장 사정으로 상품 제공 불가 시 100% 환불
- 품질 문제는 고객센터 접수 후 환불 또는 쿠폰 처리

## Food Safety

- 소비기한 내 상품만 등록 가능
- 냉장/냉동/상온 보관 안내 필수
- 당일섭취 권장 표시
- 알레르기 유발 가능성 표시
- 정육, 반찬, 도시락은 픽업시간과 보관 안내 강화

## Mistake Prevention

- 주문 발생 시 자동 재고 차감
- 수량 0개 시 자동 품절
- 픽업시간 종료 시 자동 판매 종료
- 소비자에게 픽업 알림 발송
- 판매자에게 주문/픽업 알림 발송
