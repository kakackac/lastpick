# MVP Roadmap

## Phase 0: Repository and Planning

- GitHub repository setup
- Product docs
- MVP scope
- Database draft
- Initial app/API scaffolds

## Phase 1: Manual Pilot

- 한 지역 선정
- 베이커리/반찬가게 20~30곳 리스트업
- 판매자 인터뷰
- 첫 마감박스 수동 등록 지원
- 소비자 랜딩 또는 MVP 앱으로 첫 거래 검증

## Phase 2: MVP Product

- 소비자 앱 홈/상세/결제/QR 픽업
- 판매자 앱 등록/주문확인/정산내역
- 관리자 웹 입점/주문/정산/CS
- 결제 연동
- 알림 연동

## Phase 3: Growth

- 어제와 동일하게 등록
- 판매자 템플릿
- 리뷰/재구매
- 지역별 인기 시간 분석
- 매장별 추가 매출 리포트

## Phase 4: Partnerships

- 상인회/지자체 제휴
- POS 업체 제휴
- 동네마트 체인 제휴
- 프랜차이즈 PoC
- 편의점 본사 PoC
