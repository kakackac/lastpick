# Product Requirements

## Product Name

가칭: 오늘마감, 마감박스, 세이브박스

## Problem

동네 신선식품 판매자는 매일 남는 상품을 쉽게 현금화할 채널이 부족합니다. 소비자는 가까운 곳에서 안전하고 저렴한 식품을 찾고 싶지만, 마감임박 상품 정보는 흩어져 있습니다.

## Solution

판매자가 마감임박 상품을 `마감박스`로 등록하면 소비자가 앱에서 선결제하고 지정 시간에 픽업하는 플랫폼을 제공합니다.

## MVP Scope

### Consumer App

- 오늘 픽업 가능한 마감박스 리스트
- 카테고리 필터
- 거리/가격/픽업시간 필터
- 상품 상세
- 앱 선결제
- QR 또는 주문번호 픽업
- 주문내역
- 리뷰

### Seller App

- 오늘 판매 현황
- 마감박스 등록
- 어제와 동일하게 등록
- 주문/픽업 확인
- 정산 내역
- 템플릿 관리

### Admin Web

- 입점 매장 관리
- 상품/주문 관리
- 결제/환불 관리
- 정산 관리
- CS 관리
- 지역별 거래량 분석

## Non-Goals for MVP

- 전국 확장
- 배달 기능
- 편의점 본사 API 연동
- 복잡한 AI 가격 추천
- POS 자동 연동

## Success Metrics

- 한 동네 입점 매장 20~30곳
- 매일 등록되는 마감박스 30개 이상
- 판매자 등록 시간 30초 이내
- 픽업 완료율 95% 이상
- 첫 구매 후 30일 내 재구매율 30% 이상
